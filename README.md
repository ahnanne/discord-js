# discord-js
👾 Discord.js를 이용하여 디스코드 봇 Teammy 개발하기

## 사용 기술 스택
- JavaScript

- Node.js

- Discord.js

- Heroku

## Teammy 주요 기능

- ✨ 랜덤 팀짜기 기능

- 관리자 기능

  - 유저 추방 및 차단 기능

  - 메시지 일괄 삭제 기능
